# FreshNest

The **FreshNest** is a full-stack web application developed using the **PERN Stack (PostgreSQL Express.js, React.js, Node.js)** with **AWS** integration for image storage.  
It is designed to connect customers and sellers through a simple, fast, and secure online grocery shopping experience.

---

##  1. Project Overview

In the modern era of e-commerce, grocery delivery systems have become essential for providing convenience to users and digital opportunities for store owners.  
This project aims to develop an online grocery delivery system that allows:

- **Customers (Buyers)** to browse, search, and purchase grocery items conveniently from their homes.  
- **Sellers (Vendors)** to list, manage, and fulfill orders using an web dashboard.  

The system provides a clean user interface and order management.

---

##  2. Project Objective

The main goal of FreshNest is to **bridge the gap between grocery store owners and end consumers** by offering a centralized, user-friendly digital marketplace.

**Objectives include:**

1. Creating a smooth online shopping experience for customers.  
2. Allowing sellers to easily manage their products and inventory.  
3. Providing a reliable cart, checkout, and order system.  
4. Ensuring secure login and role-based authorization.  
5. Storing media on AWS for performance and scalability.
